Stay Strong - PWA Package
===========================

This is a Progressive Web App (PWA) that can be installed on Android like a native app.

INSTALLATION INSTRUCTIONS:
-------------------------

Method 1: Direct Browser Install (Recommended)
1. Extract this ZIP file to a folder
2. Upload all files to a web server or hosting service (GitHub Pages, Netlify, Vercel, etc.)
3. Open the URL in Chrome on your Android phone
4. Chrome will show an "Install" prompt - tap it
5. The app will appear on your home screen like a native app

Method 2: Local File (Limited)
1. Extract this ZIP file
2. Open index.html in Chrome on Android
3. Tap the menu (3 dots) → "Add to Home screen"
4. Note: Some features may not work when opened as a local file

Method 3: Convert to APK (Advanced)
1. Go to https://www.pwabuilder.com
2. Enter your hosted URL
3. Follow the steps to generate an APK
4. Download and install the APK on your Android phone

FEATURES:
---------
- Works offline after first load
- Installable as a standalone app
- Local data storage (persists across sessions)
- Responsive design for all screen sizes
- PWA install prompt for easy installation

FILES INCLUDED:
---------------
- index.html      : Main app file
- manifest.json   : PWA manifest (app configuration)
- sw.js           : Service Worker (offline support)
- icons/          : App icons in various sizes
- README.txt      : This file
